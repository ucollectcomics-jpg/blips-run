Blip's Run - PC + Phone Version

Includes:
- Blip's Run loading page
- Custom Blip logo
- Loading bar from 0% to 100%
- PC keyboard controls
- Phone/touch controls
- Looping Sugar Jump music
- Existing gameplay preserved

Run on PC:
1. Unzip the folder.
2. Double-click RUN_GAME.bat.

Phone/tablet:
Open index.html in a mobile browser or upload the full folder to your web host.

Controls:
PC:
Right Arrow = run forward
Left Arrow = move back
Up Arrow = jump
Down Arrow = duck
Space = restart

Phone:
Use the on-screen arrow buttons.

Restart fix:
- Space bar and all direction buttons now restart the game after game over.
